# React TodoList

Componente de lista de tareas (Todo List) construido con React.

![Demo de TodoList](./EvidenciaToDoList.gif)

## Características

- Gestión de tareas con estado usando `useState`
- Seguimiento de última modificación con `useEffect`
- Componentes funcionales con React Hooks
- Formulario para añadir nuevas tareas
- Interfaz intuitiva con checkboxes para marcar tareas completadas

## Tecnologías

- React 18
- Vite
- JavaScript (JSX)

## Funcionalidades

- ✅ Marcar/desmarcar tareas como completadas
- ✅ Añadir nuevas tareas dinámicamente
- ✅ Timestamp automático de última modificación
- ✅ Validación de formularios (no permite tareas vacías)

## Instalación y uso

```bash
npm install
npm run dev
```

Luego abre [http://localhost:5173](http://localhost:5173) en tu navegador.
